# AutoLiqi

每天12:00（UTF+8）自动获取`code.js`和`liqi.json`，并将`liqi.json`转为`liqi.proto`和`liqi.desc`。  
Automatically fetch `code.js` and `liqi.json`, convert `liqi.json` to `liqi.proto` and `liqi.desc` every day at 12:00 (UTF+8).

## 使用方法

在[Release](https://github.com/Avenshy/AutoLiqi/releases/latest)中下载。
